# Zteam

**Краткое описание:** Android-приложение для просмотра каталога игр — поиск, фильтры, детали, рейтинги.

## Основное
- **Retrofit** для API (Rawg.io)
- **База данных** Room для оффлайн-кэша
- **Jetpack Compose** UI, MVVM архитектура
- **Поиск и фильтры** по жанрам, платформам, рейтингу
- **Детали игры** скриншоты, описание, харакатеристики

**Статус:** Не завершено. Нет избранного/отзывов/авторизации.
